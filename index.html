Core entity extensions required for the Procurement Module. Existing entities (`ASSET`, `MODEL`, `SPARE_PART`) are linked via foreign keys.

Field Type Notes

req_noVARCHARAuto-gen (PRQ-001)

total_est_valueDECIMALDrives Approval matrix

Field Type Notes
idUUIDPK

approver_idUUIDFK -> PERSONNEL
